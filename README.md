# ShedlerHouse — дипломный сайт интернет-магазина модульных домов

Готовый учебный проект на **Flask + SQLite** по теме:

**«Разработка интернет-магазина по продаже модульных домов для баз отдыха (на примере ООО „Шедлер“)»**

## Что реализовано

### Пользовательская часть
- главная страница с баннером и преимуществами;
- каталог модульных домов;
- поиск, фильтрация и сортировка;
- страница товара с характеристиками;
- форма отправки заявки;
- страница контактов с формой обратной связи;
- страницы «О компании», «Доставка и монтаж», «Решения для бизнеса»;
- избранное;
- сравнение домов.

### Административная часть
- авторизация администратора;
- dashboard со статистикой;
- управление товарами;
- управление категориями;
- просмотр и изменение статусов заявок;
- просмотр сообщений из формы контактов;
- редактирование текстового контента сайта.

## Технологии
- **Python 3**
- **Flask** — серверная логика и маршруты
- **SQLite** — база данных
- **HTML5** — структура страниц
- **CSS3** — адаптивный дизайн
- **Jinja2** — шаблоны для генерации страниц
- **Werkzeug** — безопасное хеширование пароля администратора

## Как запустить

### 1. Установить зависимости
```bash
pip install -r requirements.txt
```

### 2. Запустить проект
```bash
python app.py
```

### 3. Открыть в браузере
```text
http://127.0.0.1:5000
```

## Что можно написать в дипломе
В проекте реализован интернет-магазин каталожного типа, ориентированный на продажу модульных домов для баз отдыха. Пользователь может просматривать модели домов, изучать их характеристики, сравнивать предложения, добавлять товары в избранное и отправлять заявку менеджеру. Администратор может управлять товарами, категориями, заявками и текстовым наполнением сайта.
